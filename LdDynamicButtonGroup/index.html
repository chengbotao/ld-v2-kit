# LdDynamicButtonGroup 动态按钮组

LdDynamicButtonGroup 是一个基于 Element UI 按钮组件封装的动态按钮组组件，支持自动将超出数量的按钮收纳到下拉菜单中。

# 功能特性

  • ✅ 自动管理按钮显示数量
  • ✅ 超出部分显示在下拉菜单中
  • ✅ 支持自定义下拉按钮样式
  • ✅ 支持为每个按钮添加插槽
  • ✅ 支持按钮公共属性配置
  • ✅ 支持下拉菜单属性配置

# 基本用法

默认最多显示3个按钮，超出部分会显示在下拉菜单中

<template>
  <div>
    <ld-dynamic-button-group
      :buttonList="buttonList"
      @click="handleClick"
    />
    <div style="margin-top: 20px;">
      <h4>点击结果：{{ clickedButton ? clickedButton.label : '未点击' }}</h4>
    </div>
  </div>
</template>

<script>
export default {
  data() {
    return {
      buttonList: [
        { key: 'btn1', label: '按钮1', props: { type: 'primary' } },
        { key: 'btn2', label: '按钮2', props: { type: 'success' } },
        { key: 'btn3', label: '按钮3', props: { type: 'warning' } },
        { key: 'btn4', label: '按钮4', props: { type: 'danger' } },
        { key: 'btn5', label: '按钮5', props: { type: 'info' } }
      ],
      clickedButton: null
    };
  },
  methods: {
    handleClick(button) {
      this.clickedButton = button;
      console.log('点击了按钮:', button);
    }
  }
};
</script>

# 自定义下拉按钮

通过 dropdown 插槽自定义下拉按钮的样式和内容

<template>
  <div>
    <ld-dynamic-button-group
      :buttonList="buttonList"
      :maxVisibleButtons="2"
      @click="handleClick"
    >
      <template #dropdown>
        <i class="el-icon-more"></i>
      </template>
    </ld-dynamic-button-group>
  </div>
</template>

<script>
export default {
  data() {
    return {
      buttonList: [
        { key: "btn1", label: "编辑", props: { type: "primary" } },
        { key: "btn2", label: "删除", props: { type: "danger" } },
        { key: "btn3", label: "复制", props: { type: "warning" } },
        { key: "btn4", label: "导出", props: { type: "success" } },
      ],
    };
  },
  methods: {
    handleClick(button) {
      console.log("点击了按钮:", button);
    },
  },
};
</script>

# 使用插槽

通过插槽为每个按钮自定义内容，可添加图标等

<template>
  <div>
    <ld-dynamic-button-group
      :buttonList="buttonList"
      @click="handleClick"
    >
      <!-- 自定义按钮1的内容 -->
      <template #btn1="{ scope }">
        <i class="el-icon-edit"></i>
        <span style="margin-left: 5px;">{{ scope.label }}</span>
      </template>
      
      <!-- 自定义按钮2的内容 -->
      <template #btn2="{ scope }">
        <i class="el-icon-delete"></i>
        <span style="margin-left: 5px;">{{ scope.label }}</span>
      </template>
      
      <!-- 自定义按钮3的内容 -->
      <template #btn3="{ scope }">
        <i class="el-icon-copy-document"></i>
        <span style="margin-left: 5px;">{{ scope.label }}</span>
      </template>
    </ld-dynamic-button-group>
  </div>
</template>

<script>
export default {
  data() {
    return {
      buttonList: [
        { key: 'btn1', label: '编辑', props: { type: 'primary' } },
        { key: 'btn2', label: '删除', props: { type: 'danger' } },
        { key: 'btn3', label: '复制', props: { type: 'warning' } },
        { key: 'btn4', label: '导出', props: { type: 'success' } },
        { key: 'btn5', label: '打印', props: { type: 'info' } }
      ]
    };
  },
  methods: {
    handleClick(button) {
      console.log('点击了按钮:', button);
    }
  }
};
</script>

# API

# Props

参数 说明 类型 默认值 可选值
buttonList 按钮配置数组 Array [] -
maxVisibleButtons 最多显示的按钮数量 Number 3 -
buttonProps 按钮公共属性 Object {} -
dropdownProps 下拉菜单属性 Object {} -

# buttonList 配置项

参数 说明 类型 默认值 可选值
key 按钮唯一标识 String - -
label 按钮文本 String - -
props 按钮属性 Object {} -

# Events

事件名称 说明 回调参数
click 点击按钮时触发 button: 按钮配置对象

# Slots

插槽名称 说明
dropdown 自定义下拉按钮插槽
[key] 自定义按钮内容插槽，[key] 为按钮的 key 值

# 注意事项

  1. 每个按钮必须提供唯一的 key 值，用于插槽命名和元素标识
  2. 当 buttonList 为空数组时，不会显示任何内容
  3. 当按钮数量未超过 maxVisibleButtons 时，不会显示下拉按钮
